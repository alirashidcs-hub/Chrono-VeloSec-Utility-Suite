# 🚀 Chrono VeloSec Utility Suite

A comprehensive **C++ console-based desktop application** developed as a semester project for the **Object-Oriented Programming (OOP)** course at the **University of Engineering and Technology (UET) Taxila**.

This project combines multiple productivity and security tools into a single application while demonstrating the practical implementation of Object-Oriented Programming concepts, modular programming, and efficient software design.

---

## 📖 Overview

Chrono VeloSec Utility Suite was designed to provide several useful utilities within one console application. The project emphasizes clean code structure, reusable components, and user-friendly interaction while strengthening core C++ programming skills.

It serves as a practical demonstration of applying OOP principles to develop a maintainable and scalable software solution.

---

## ✨ Features

- ⌨️ Typing Speed Tester
- 🔐 Password Strength Analyzer
- ⏱ Stopwatch
- ⏲ Countdown Timer
- 🔑 Random Password Generator
- 📋 Menu-Driven Interface
- 🧩 Modular Code Structure
- 💻 User-Friendly Console Experience
