# Chrono VeloSec Utility Suite

## Overview
Chrono VeloSec Utility Suite is a C++ desktop application developed using Object-Oriented Programming concepts. It provides multiple productivity and security tools in a single application.

## Features
- Typing Speed Test
- Password Strength Analyzer
- Stopwatch
- Timer
- User-friendly Console Interface
- Modular OOP Design

## Technologies
- C++
- Object-Oriented Programming
- STL
